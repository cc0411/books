[python sh库](http://amoffat.github.io/sh/)

sh is a full-fledged subprocess replacement for Python 2.6 - 3.5, PyPy and PyPy3 that allows you to call any program as if it were a function:
安装
```
pip install sh
```

#### 2. 异常捕获

```
try:
    sh.ls("/doesnt/exist")
except sh.ErrorReturnCode_2:
    print("directory doesn't exist")
```

#### 5. bake命令命令参数
```
In [24]: import sh

In [25]: du=sh.du.bake('-shc')

In [26]: print (du('/tmp/yun'))
6.3M	/tmp/yun
6.3M	总用量
```
#### 6. glob列出文件
```
In [27]: import sh

In [28]: list=sh.glob('/tmp/yun*')

In [29]: print(list)

```
