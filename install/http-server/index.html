  • 3. 发布至自己的HTTP Server

通过mkdocs build编译出html并手动同步至http server的根目录

生成站点文件¶

在git根目录下执行命令

mkdocs build

命令执行完毕后可以看到site目录

发布至http server¶

将site目录里的所有东西同步至自己的http server的根目录
